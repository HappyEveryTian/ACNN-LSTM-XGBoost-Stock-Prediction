# 基于注意力机制的CNN-LSTM和XGBoost混合模型预测分析-股票预测

## Requirements

代码在Python3.7.4版本下运行，以下是各个依赖版本：
```
numpy==1.21.6
sklearn==0.21.3
statsmodels==0.10.1
pandas==1.2.1
tensorflow==2.1.0
keras==2.3.1
xgboost==1.5.0
matplotlib==3.5.3
tushare==1.4.19
```
你也可以根据requirements.txt进行快速安装。
